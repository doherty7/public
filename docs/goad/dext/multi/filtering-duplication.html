중복된 파일을 제한하는 예제입니다.
설명

동일한 로컬 파일 등록을 허용할지 여부를 결정할 수 있습니다.

가상 파일과 로컬 파일은 서로 비교 대상이 아니므로, 가상 파일과 로컬 파일은 중복이 될 수 있습니다.

multi 버전을 사용하는 경우, 로컬 자원의 경로 검사를 수행할 수 없기 때문에, 중복 비교 대상은 파일명과 크기 그리고 파일의 최종 수정 일자로 비교를 합니다. 

var dx = dx5.get("컴포넌트 아이디");

// 중복을 허용합니다.
var filtered = dx.checkDuplication(false);
