DEXTUploadX5 Basic Samples (with DEXTUploadNJ)

DEXTUploadX5 전반적인 기능을 이해하기 위해 작성된 샘플입니다.

샘플을 실행하기 전에 아래 내용을 반드시 확인하길 바랍니다.

  • DEXTUploadX5는 웹브라우저에서 동작하는 클라이언트 컴포넌트입니다.
  • DEXTUploadX5는 HTML5 기반으로 동작하는 멀티 버전과 ActiveX 기반으로 동작하는 IE 버전이 있습니다.
  • 해당 샘플은 DEXTUploadX5가 업로드한 파일을 처리하기 위한 서버 측 컴포넌트로 DEXTUploadNJ가 사용되고 있습니다.
  • DEXTUploadX5를 사용하려면 평가판 키가 필요합니다.
  • DEXTUploadX5 평가판을 다운로드할 때 이메일로 제공된 평가판 키를 웹루트/dx5/dextuploadx5-configuration.js 파일에 설정하시길 바랍니다.
  • DEXTUploadNJ는 평가판 라이선스로 설정되어있으므로 별도로 설정할 필요가 없습니다.(DEXTUploadX5와 동일하게 30일 사용 가능)
  • WEB-INF/dextuploadnj.config 파일은 DEXTUploadNJ의 평가판 라이선스 파일입니다.
  • 샘플은 Java SE 1.6, Servlet API 2.5, Spring Framework 3.0.5, Maven 환경을 기반으로 제작되었습니다. => java 1.8 / spring 4.2.4.RELEASE - 전자정부프레임웍 / servlet-api 2.5
  • 샘플은 UTF-8 환경을 기준으로 합니다. 개발환경이 Ansi인 경우 한글 주석이 모두 깨져 보일 수 있습니다.
