isHDRunning

최소지원버전
2.3.5.0
지원환경
멀티 모듈, IE 모듈 모두 지원
형식
dx5.isHDRunning(success, fail);
파라미터
이름 형식 설명
success function

HD 응용프로그램이 실행 중이면 호출되는 콜백 함수입니다.
fail function

HD 응용프로그램이 실행 중이 아니라면 호출되는 콜백 함수입니다.
설명

HD 응용프로그램이 현재 실행 중인지를 확인할 때 사용하는 함수입니다.

함수를 호출하여 성공적인 응답을 얻는다면 success 파라미터로 등록된 콜백함수가 호출이 되며, 실패한다면 fail 파라미터로 등록된 콜백함수가 호출됩니다.

function checkHDRunning() {
    dx5.isHDRunning(function (ver) {
        // 성공하면 버전을 나타내는 문자열을 파라미터로 얻을 수 있습니다.
        alert(ver);
    }, function (code, msg) {
        // 실패하면 실패 코드와 메시지를 얻을 수 있습니다.
        alert(code + ": " + msg);
    });
}

isHDRunning 함수는 현재 HD 응용프로그램의 실행 여부만을 확인합니다. 그러므로 HD 응용프로그램의 설치 여부는 알 수는 없습니다.

응답이 실패하는 경우 브라우저 콘솔에는 다음과 같은 추가 오류 정보가 출력될 수 있으며, HD 설치 여부에 따라 메시지가 달라질 수 있습니다.
