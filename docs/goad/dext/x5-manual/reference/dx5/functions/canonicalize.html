canonicalize

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 모두 지원
형식
fullURL = dx5.canonicalize(relativeURL);
파라미터
이름 형식 설명
relativeURL String

URL("/", "./", "../"로 시작하는)입니다.
설명

파라미터로 주어진 경로로부터 전체 경로를 반환합니다.

DEXTUploadX5 컴포넌트와는 상관없이 샘플 작업을 하기 위해서 만들어진 일종의 유틸리티 기능입니다.

/, ./ 또는 ../으로 시작하는 경로를 전체경로로 변경하여 반환합니다.

잘못된 경로 값이 주어지는 경우, 올바르지 못한 전체 경로가 반환될 수 있으니, 참고 용도로만 사용하기를 권장합니다.

// 현재 URL이 http://abc.com/path1/path2/current.html 라면
var full = dx5.canonicalize("../abc/resource.txt");
// full 변수는 http://abc.com/path1/abc/resource.txt 가 됩니다.
