browser Object

최소지원버전
1.1.0.0
지원환경
멀티 모듈, IE 모듈 모두 지원
설명

dx5.browser 객체는 현재 페이지가 어떤 브라우저에서 로드되었는지를 판단할 수 있는 속성을 가지고 있습니다.

컴포넌트를 생성하기 전에 객체가 생성이 되므로, 브라우저에 따라 멀티 모듈과 IE 모듈을 구별하여 사용할 때 유용합니다.

// ActiveX가 지원되는 브라우저인지를 확인합니다.
var supportActiveX = dx5.browser.isIE;
if (supportActiveX) {
  dx5.create({ mode: "ie", ... });
} else {
  dx5.create({ mode: "multi", ... });
}
이름 형식 설명
isOpera Boolean 오페라 브라우저이면 true입니다.
isFirefox Boolean 파이어폭스 브라우저이면 true입니다.
isChrome Boolean 크롬 브라우저이면 true입니다.
isIE Boolean Internet Explorer 브라우저이면 true입니다.
isEgde Boolean Microsoft Egde 브라우저이면 true입니다.
isUnKnown Boolean 그 밖의 브라우저이거나, 알 수 없으면 true입니다.

최근 브라우저가 아닌 오래된 브라우저의 경우 올바르게 인식 못할 수 있습니다.
