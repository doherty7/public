onDX5ItemsAdded

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
function onDX5ItemsAdded(id, count) { ... }
파라미터
이름 형식 설명
id String 이벤트가 발생한 컴포넌트 아이디
count Number 등록된 항목의 개수
반환 값
설명

onDX5ItemsAdded 함수는 등록이 완료된 후 호출되는 콜백 함수입니다.

function onDX5ItemsAdded(id, count) {
  alert(count + "개 항목이 등록되었습니다.");
}
