onDX5CompressCompleted

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
function onDX5CompressCompleted(id) { ... }
파라미터
이름 형식 설명
id String 이벤트가 발생한 컴포넌트 아이디
반환 값
설명

onDX5CompressCompleted 함수는 압축 작업이 완료되면 호출되는 콜백 함수입니다.

function onDX5CompressCompleted(id) {
  alert("압축 작업이 완료되었습니다.");
}

onDX5CompressCompleted 함수는 IE 모듈에서 로컬 파일을 압축하는 과정이 완료되면 호출됩니다. 멀티 모듈(1.3.0.0 이상)에서 압축과 관련된 onDX5CompressWaitingBegin, onDX5CompressWaitingCompleted, onDX5CompressWaitingStopped 콜백 함수를 지원합니다. 이 함수는 서버에 압축을 요청하고 대상을 다운로드하는 과정에서 호출되는 함수들로써, IE 모듈의 로컬 압축 기능과는 전혀 상관이 없음을 주의하십시오.
