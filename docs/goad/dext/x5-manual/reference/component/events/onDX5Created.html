onDX5Created

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
function onDX5Created(id) { ... }
파라미터
이름 형식 설명
id String 이벤트가 발생한 컴포넌트 아이디
반환 값
설명

onDX5Created 함수는 컴포넌트가 생성되면 호출되는 콜백 함수입니다.

DEXTUploadX5는 DOM의 onload 이벤트를 가지고 생성 주기를 판단할 수 없습니다.

컴포넌트가 성공적으로 생성이 되어야 여러 가지 추가 설정을 할 수 있기 때문에 onDX5Created 콜백 함수를 호출하는 생성 이벤트를 제공하고 있습니다.

function onDX5Created(id) {
  // 업로드 경로 설정
  // 스타일 설정
  // 가상 파일 등록
}
