setUploadBlockSize

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.setUploadBlockSize(blockSize);
파라미터
이름 형식 설명
blockSize Number 블록 크기, 바이트 단위로 설정합니다.
반환 값
설명

대용량 파일 업로드를 수행할 때, 업로드 단위 크기를 설정합니다.

DEXTUploadX5는 대용량 파일 업로드를 수행할 때, 파일을 주어진 블록 크기로 나누어 전송을 합니다.

100MB 파일을 업로드할 때, 블록 크기가 10MB면 메타 데이터 전송을 포함하여 총 11번(헤더전송 1번 + 파일전송 10번) 요청이 발생합니다.

setUploadBlockSize 함수는 대용량 업로드에서만 동작하기 때문에, "ORAF", "OROF" 전송에서는 무시됩니다.

function onDX5Created(id) {
  var dx = dx5.get(id);
  // 대용량 업로드 방식으로 사용합니다.
  dx.setUploadMode("EXNJ");
  // 업로드 블록 단위를 10MB로 설정합니다.
  dx.setUploadBlockSize(10 * 1024 * 1024);
}
