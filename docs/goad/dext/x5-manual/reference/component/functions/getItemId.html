getItemId

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
id = object.getItemId(index);
파라미터
이름 형식 설명
index Number 항목의 순서
반환 값
주어진 순서에 해당하는 항목의 고유 아이디
설명

주어진 순서에 해당하는 항목의 고유 아이디를 반환합니다.

// 네 번째 항목의 고유 아이디 값을 반환합니다.
id = dx.getItemId(3);
// 3을 반환합니다.
index = dx.getItemIndex(id);
