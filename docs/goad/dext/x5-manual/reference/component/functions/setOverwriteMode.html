setOverwriteMode

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
success = object.setOverwriteMode(mode);
파라미터
이름 형식 설명
mode String

다운로드 중복 처리 옵션

값 설명
"QUESTION" 다운로드 위치에 중복 파일이 있을 때마다, 처리 방법을 질의합니다.
"OVERWRITE" 중복 파일을 덮어씁니다.
"RENAME" 새 이름으로 변경합니다. 뒤에 숫자가 붙습니다. [0], [1], ...
"SKIP" 내려받지 않고 건너뜁니다.
반환 값
옵션이 성공적으로 설정되었다면 true, 아니면 false
설명

다운로드 중복 처리 옵션을 설정합니다.

다운로드 경로에 이미 대상 파일이 있을 때, 컴포넌트가 파일을 처리하는 방법을 설정할 수 있습니다.

기본값은 "QUESTION"이며, 사용자에게 중복 처리 방법을 질의합니다.

중복 파일이란 이름이 동일한 파일을 의미합니다.

// 중복이 발견되면, 다운로드 받은 새 파일로 덮어씁니다.
dx.setOverwriteMode("OVERWRITE");
console.log(dx.getOverwriteMode());
