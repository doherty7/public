getTotalLocalFileCount

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
count = object.getTotalLocalFolderCount();
파라미터
반환 값
로컬 폴더의 개수
설명

로컬 폴더의 개수를 반환합니다.

DEXTUploadX5가 가질 수 있는 항목은 로컬 파일, 로컬 폴더, 가상 파일이 있습니다.

멀티 모듈은 로컬 파일, 가상 파일을 다룰 수 있으며, IE 모듈은 로컬 폴더가 항목으로 포함됩니다.

로컬 폴더는 컴포넌트로부터 얻을 수 있는 항목의 type 속성이 "FOLDER"입니다.

로컬 폴더는 폴더 선택 다이얼로그를 통해서 추가할 수 있습니다.

// 로컬 폴더의 개수를 출력합니다.
console.log(dx.getTotalLocalFolderCount());
