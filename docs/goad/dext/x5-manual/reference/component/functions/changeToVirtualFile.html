changeToVirtualFile

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
// 멀티 모듈
object.changeToVirtualFile(id, openUrl, downUrl);
// IE 모듈
object.changeToVirtualFile(id, url);
파라미터
이름 지원 형식 설명
id 멀티, IE String 로컬 자원 항목의 고유 아이디
openUrl 멀티 String

파일을 열 때 사용되는 URL, 스키마(http, https)를 포함하는 전체 주소이어야 합니다.

파일을 열 URL이 없다면, 빈 문자열을 입력합니다.
downUrl 멀티 String

파일을 다운로드할 때 사용되는 URL, 스키마(http, https)를 포함하는 전체 주소이어야 합니다.

파일을 다운로드할 URL이 없다면, 빈 문자열을 입력합니다.
url IE String

파일을 다운로드하거나 열 때 사용되는 URL, 스키마(http, https)를 포함하는 전체 주소이어야 합니다.

파일을 다운로드하거나 열 URL이 없다면, 빈 문자열을 입력합니다.
반환 값
설명

로컬 자원 항목을 가상 파일로 변경합니다.

로컬 자원을 가상 파일로 변경하여 사용할 수도 있습니다.

반대로 가상 파일을 로컬 파일로 변경하는 기능은 제공하지 않습니다.

// 로컬 파일 항목을 가상 파일 항목으로 변경합니다.
dx.changeToVirtualFile(id, "http://.../...", "http://../...");
// 다운로드 혹은 열기 URL 정보 없이 사용하는 경우 빈 문자열을 입력합니다.
dx.changeToVirtualFile(id, "", "");

// IE 모듈은 openUrl, downUrl을 구분하지 않고 하나로 처리가 가능합니다.
dx.changeToVirtualFile(id, "http://.../...");
dx.changeToVirtualFile(id, "");
