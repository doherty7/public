setPreviewEnable

최소지원버전
1.0.0.0
지원환경
멀티 모듈(1.2.0.0 이상), IE 모듈 지원
형식
object.setPreviewEnable(enable);
파라미터
이름 형식 설명
enable Boolean 미리보기 뷰어를 사용하려면 true, 아니면 false
반환 값
설명

미리보기 뷰어 사용 여부를 설정합니다.

대상이 이미지 파일이고, 미리보기가 가능한 경우 화면 또는 이벤트로 미리보기 기능을 제공합니다.

지원하는 포맷은 다음과 같습니다.

  • 멀티 모듈: JPG, GIF, PNG, BMP
  • IE 모듈: JPG, GIF, PNG, BMP(내장뷰어 추가 포맷: ICON, WMF, EMF, TIFF)
// true로 설정하면 뷰어를 사용할 수 있습니다.
dx.setPreviewEnable(true);

// 배경색을 설정합니다.
dx.setPreviewBackgroundColor("#ff0000");

// 내장 뷰어를 사용하여 미리보기를 수행하도록 설정합니다. (기본값 1)
dx.setPreviewMethod(1);

// 첫 번째 파일이 로컬 이미지 파일이라면 내장 뷰어를 실행합니다.
dx.preview(0);
