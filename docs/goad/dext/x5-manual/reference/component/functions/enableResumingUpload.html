enableResumingUpload

최소지원버전
1.0.0.0
지원환경
멀티 모듈 지원
형식
object.enableResumingUpload(enable);
파라미터
이름 형식 설명
enable Boolean true이면 이어올리기를 사용합니다. 기본값은 true입니다.
반환 값
설명

대용량 업로드를 수행할 때, 이어올리기를 사용할 지 여부를 설정합니다.

IE 모듈과 달리 멀티 모듈은 파일의 고유성(파일의 이름, 크기, 최종 수정일)을 경로로 구별하지 못합니다.

그러므로 서버와의 연결을 완전히 끊은 후, 다시 업로드되는 과정에서 이어 올릴 대상을, 파일의 이름과 크기 그리고 수정 일자로만 매치시킬 수 있습니다.

만일 이런 판별 방식이 확신이 없다면, false로 설정하여 이어올리기 기능은 사용하지 않도록 합니다.

멀티 모듈과 달리 IE 모듈은 경로 기반으로 대상을 찾기 때문에, 무조건 이어올리기로 동작합니다.

enableResumingUpload 함수는 대용량 업로드에서만 동작하기 때문에, "ORAF", "OROF" 전송에서는 무시되며, 대용량 파일 업로드를 제외한, "ORAF", "OROF"에서는 멀티 모듈, IE 모듈 모두 이어올리기 기능이 제공되지 않습니다.

function onDX5Created(id) {
  var dx = dx5.get(id);
  // 대용량 업로드 방식으로 사용합니다.
  dx.setUploadMode("EXNJ");
  // 업로드 블록 단위를 10MB로 설정합니다.
  dx.setUploadBlockSize(10 * 1024 * 1024);
  // 이어올리기를 사용하지 않습니다.
  dx.enableResumingUpload(false);
}
