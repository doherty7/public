getWithCredentials

최소지원버전
2.2.0.0
지원환경
멀티 모듈
형식
isWith = object.getWithCredentials();
파라미터
반환 값
자격 정보를 사용할 수 있다면 true, 아니면 false입니다.
설명

쿠키, 인증 헤더, TLS 클라이언트 자격 증명과 같은 자격 정보를 가지고 서버로 요청을 시도할 수 있는지 여부를 반환합니다.

function onDX5Created(id) {
  var dx = dx5.get(id);
  console.log(dx.getWithCredentials());
}
