setUploadableFilesFrom

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.setUploadableFilesFrom(otherComponent);
파라미터
이름 형식 설명
otherComponent Object 업로드할 파일을 가져올 DEXTUploadX5 컴포넌트 객체
반환 값
설명

특정 컴포넌트가 가진 로컬 자원을 현재 컴포넌트에 추가합니다.

DEXTUploadX5는 파일 선택 다이얼로그, 폴더 선택 다이얼로그만을 가지고 로컬 자원을 추가할 수 있도록 설계되었습니다.

그러나 컴포넌트 끼리 항목을 주고 받아야 하는 상황에서 로컬 자원 데이터를 서로 공유해야 하는 상황이 발생합니다.

setUploadableFilesFrom 함수는 스크립트에서 로컬 자원 항목을 등록할 수 있도록 생성된 함수이며, 오로지 컴포넌트끼리 항목을 주고받을 목적으로 만들어졌습니다.

// 멀티 모듈에서 "dext5-uploader" 컴포넌트가 
// "dext5-a", "dext5-b" 컴포넌트의 로컬 자원 항목을 대신 업로드하는 방법입니다.

var uploader = dx5.get("dext5-uploader");
			
uploader.clearItems();

// 첫 번째 컴포넌트로부터 업로드를 하기 위한 파일 정보를 얻습니다.
uploader.setUploadableFilesFrom(dx5.get("dext5-a"));
// 두 번째 컴포넌트로부터 업로드를 하기 위한 파일 정보를 얻습니다.
uploader.setUploadableFilesFrom(dx5.get("dext5-b"));

uploader.setUploadURL("업로드 경로");
uploader.upload("AUTO");

setUploadableFilesFrom 함수를 사용하여 파일을 얻은 컴포넌트는 IE 모듈에서는 화면에 항목이 그려지지만, 멀티 모듈에서는 UI 변화가 없습니다.

2.3.1.0 버전 이전에는 함수명이 setUploableFilesFrom이었습니다.
