getItems

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
items = object.getItems([onlyLocal, isOrign]);
파라미터
이름 지원 형식 설명
onlyLocal 멀티, IE Boolean [옵션] true이면 로컬 자원만 반환합니다.
isOrign 멀티 Boolean [옵션] 반환된 배열의 요소가 복사본이 아닌 원본이 필요하면 true로 설정합니다.
반환 값
항목 배열
설명

항목을 배열로 반환합니다.

DEXTUploadX5가 가질 수 있는 항목은 로컬 파일, 로컬 폴더, 가상 파일이 있습니다.

멀티 모듈은 로컬 파일, 가상 파일을 다룰 수 있으며, IE 모듈은 로컬 폴더가 항목으로 포함됩니다.

// 모든 항목을 갖는 배열을 반환합니다. (멀티, IE 공통)
var all = dx.getItems();
// 로컬 항목만을 갖는 배열을 반환합니다. (멀티, IE 공통)
var locals = dx.getItems(true);
// 모든 원본 항목을 갖는 배열을 반환합니다. (멀티 모듈만)
var origins = dx.getItems(false, true);

isOrigin 파라미터는 멀티 모듈 전용 옵션이며, 이 값이 true이면 복사본이 아닌 원본 항목 요소가 반환됩니다.

이 옵션은 멀티 컴포넌트 업로드에서 사용이 되며, 그 외의 경우에는 항상 복사본을 사용해야 합니다.
