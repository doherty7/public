getSupportPHP

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
enable = object.getSupportPHP();
파라미터
반환 값
PHP 환경을 지원할지 여부
설명

PHP 환경을 지원할지를 반환합니다.

DEXTUploadX5 제품은 파일을 'DEXTUploadX5_FileData' 폼 이름으로 전송합니다.

ORAF 전송 방식을 사용하는 경우, 여러 파일을 동시에 업로드하더라도 폼 이름은 동일합니다.

그러나 PHP의 경우에는 동일한 폼 이름으로 여러 파일을 동시에 올리려면 폼 이름을 배열처럼 설정해야 합니다.

즉 'DEXTUploadX5_FileData[]' 처럼 배열 형식으로 폼 이름을 지어야 합니다.

function onDX5Created(id) {
  var dx = dx5.get(id);
  // PHP라면 true로 설정합니다.
  dx.setSupportPHP(true);
}

기본 업로드 방식인 ORAF인 경우는 PHP환경임을 설정해야 합니다. 그러나 OROF, EXNJ 업로드 방식은 한 요청에 하나의 파일 정보만 전달하므로 setSupportPHP 함수를 사용할 필요가 없습니다.
