getVersion

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
strVersion = object.getVersion();
파라미터
반환 값
버전 문자열
설명

DEXTUploadX5 컴포넌트 버전을 반환합니다.

getVersion 함수를 통하여 얻을 수 있는 값은 제품의 실제 버전입니다.

반면에 dextuploadx5-configuration.js 파일의 version 속성은 사용하고자 하는 버전을 의미합니다.

IE 모듈에서는 설치된 제품의 버전이 사용하고자 하는 버전보다 낮으면, 다운로드 페이지를 열지만,

멀티 모듈은 사용하고자 하는 버전에 아무런 영향을 받지 않습니다.

function onDX5Created(id) {
  var dx = dx5.get(id);
  // 버전을 출력합니다.
  console.log(dx.getVersion());
}
