getItemIds

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
ids = object.getItemIds([onlyLocal]);
파라미터
이름 형식 설명
onlyLocal Boolean [옵션] true이면 로컬 자원만 대상으로 삼습니다.
반환 값
모든 항목의 고유 아이디 값을 요소로 갖는 배열
설명

모든 항목의 고유 아이디 값을 요소로 갖는 배열을 반환합니다.

파라미터 값이 true이면 가상 파일을 제외한 로컬 자원(파일, 폴더)만 카운트된 값이 반환됩니다.

DEXTUploadX5가 가질 수 있는 항목은 로컬 파일, 로컬 폴더, 가상 파일이 있습니다.

멀티 모듈은 로컬 파일, 가상 파일을 다룰 수 있으며, IE 모듈은 로컬 폴더가 항목으로 포함됩니다.

// 모든 항목의 고유 아이디 값을 요소로 갖는 배열을 얻습니다.
ids = dx.getItemIds();
// 로컬 자원의 고유 아이디를 얻으려면 파라미터를 true로 설정합니다.
ids = dx.getItemIds(true);
