revokeVirtualFile

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.revokeVirtualFile(id);
파라미터
이름 형식 설명
id String 삭제 상태로 보관 중인 가상 파일의 고유 아이디
반환 값
설명

주어진 고유 아이디에 해당하는 삭제된 가상 파일을 복구합니다.

항목을 삭제할 때, 대상이 가상 파일이면, 바로 삭제하지 않고 삭제 상태로 보관합니다.

이와 달리 로컬 자원들은 상태 변경없이 완전히 제거됩니다.

이렇게 삭제 상태로 보관 중인 가상 파일을 다시 복구하려면 revokeVirtualFile 함수를 사용합니다.

// 주어진 아이디에 해당하는 삭제된 가상 파일을 복구합니다.
dx.revokeVirtualFile(id);
