getMaxTotalSize

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
maxTotalSize = object.getMaxTotalSize();
파라미터
반환 값
전체 파일의 최대 허용 크기(바이트), 기본값은 -1입니다.
설명

컴포넌트에 추가할 수 있는 전체 파일의 최대 허용 크기를 반환합니다.

값이 -1이면 제한을 하지 않는다는 의미를 갖습니다.

function onDX5Created(id) {
  var dx = dx5.get(id);
  // 전체 파일 용량의 합이 300MB가 넘는 경우 등록할 수 없습니다.
  dx.setMaxTotalSize(1024 * 1024 * 300);
  // 컴포넌트에 추가할 수 있는 전체 파일 허용 크기의 최댓값을 출력합니다.
  console.log(dx.getMaxTotalSize());
}

최댓값을 아무리 높게 잡아도 서버에서 받을 수 없는 크기라면 업로드가 실패합니다.

OROF, ORAF 업로드 방식의 경우 전체 파일은 허용 크기는 대략 2GB를 넘지 못합니다.

업로드할 데이터의 양이 커지면, 서버 리소스 점유율이 높아져 동시 처리가 원할하지 못할 수 있습니다.

일반적으로는 큰 데이터를 업로드할 이유가 없으므로, 가능한 용량을 제한하는 것을 권장합니다.
