setMaxFileCount

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.setMaxFileCount(count [, withVF]);
파라미터
이름 형식 설명
count Number 최대 제한 개수, 기본값은 -1입니다.
withVF Boolean [옵션] true이면 가상 파일을 포함하여 계산합니다. 기본값은 false입니다.
반환 값
설명

컴포넌트에 추가할 수 있는 파일의 최대 개수를 설정합니다.

업로드할 파일의 개수를 제한하기 위해서 추가할 수 있는 파일의 최대 개수를 제한할 수 있습니다.

값이 -1이면 제한을 하지 않는다는 의미를 갖습니다.

function onDX5Created(id) {
  var dx = dx5.get(id);
  // 파일을 최대 10까지 등록할 수 있습니다.
  dx.setMaxFileCount(10);
  // 만일 가상 파일을 포함하여 최대 10까지 등록하려면 두 번째 파라미터 값을 true로 입력합니다.
  // 즉 로컬 파일의 개수는 (10 - 가상 파일의 개수) 만큼만 등록이 가능합니다.
  dx.setMaxFileCount(10, true);
  // 컴포넌트에 추가할 수 있는 파일의 최대 개수를 출력합니다.
  console.log(dx.getMaxFileCount());
}
