clearItems

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.clearItems([flag]);
파라미터
이름 형식 설명
flag String

[옵션]

  • "FILE": 로컬 파일을 모두 제거합니다.
  • "VIRTUAL": 가상 파일을 모두 제거합니다.
  • "DVIRTUAL": 삭제된 가상 파일을 모두 제거합니다.
  • undefined: 모든 항목을 제거합니다.
반환 값
설명

항목을 제거합니다.

파라미터가 정해지지 않으면, 모든 항목을 제거합니다.

DEXTUploadX5에서 '삭제'와 '제거'는 다르게 구분합니다.

삭제하려는 대상이 가상 파일인 경우, 가상 파일은 제거가 되지 않고 내부적으로 삭제된 상태로만 관리가 됩니다.

clearItems를 사용하면, 관리없이 완전히 제거가 됩니다.

// 로컬 파일 항목을 모두 제거합니다.
dx.clearItems("FILE");
// 모든 항목을 제거합니다.
dx.clearItems();
