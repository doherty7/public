setLimitMultiDownloadSize

최소지원버전
1.1.0.0
지원환경
멀티 모듈 지원
형식
object.setLimitMultiDownloadSize(limit);
파라미터
이름 형식 설명
limit Number 제한된 다운로드 크기(바이트 단위)
반환 값
설명

다운로드 되는 파일의 개별 제한 크기를 설정합니다.

1.1.0.0 버전부터 멀티 모듈도 다중 파일 다운로드를 지원합니다. 그러나 IE 모듈과 달리 멀티 모듈의 멀티 다운로드 기능은 브라우저 성능 문제로 인하여 다운로드하는 파일의 크기에 제한을 두도록 하였습니다.

기본값은 100MB이며, 파일 크기를 그 이상 허용하려면 setLimitMultiDownloadSize 함수를 사용하여 제한된 크기를 변경하면 됩니다.

그러나 이 함수는 다운로드가 허용되는 대상을 크기를 변경하는 것이지, 브라우저의 성능에 상관없이 다운로드가 가능하도록 기능을 제공하는 것이 아닙니다. HTML5의 기능을 사용하여 파일을 연속적으로 다운로드하려면, 일시적으로 다운로드 대상을 메모리에 로드하기 때문에, 동영상 파일과 같이 용량이 큰 파일들은 멀티 모듈에서 적절한 다중 파일 다운로드 대상이 아닙니다. 그러므로 대용량 다운로드, 이어 받기와 같은 기능이 필요하다면, 멀티 모듈 대신에 IE 모듈을 사용하는 것을 권장합니다. 

// 다운로드 제한 크기를 300MB로 설정합니다. 
dx.setLimitMultiDownloadSize(1024 * 1024 * 300);
