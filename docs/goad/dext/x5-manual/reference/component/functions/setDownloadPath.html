레퍼런스 > Component Object > Methods > setDownloadPath

setDownloadPath

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
success = object.setDownloadPath(path);
파라미터
이름 형식 설명
path String

다운로드 경로

값 설명
드라이브 문자열로 시작 절대 경로
<home>로 시작 사용자 폴더 하위 경로
<doc>로 시작 문서 폴더 하위 경로
<desktop>로 시작 바탕화면 폴더 하위 경로
반환 값
다운로드 경로가 성공적으로 설정되었다면 true, 아니면 false
설명

다운로드 경로를 설정합니다.

// 드라이브 문자열로 시작하면 절대 경로가 설정됩니다.
dx.setDownloadPath("c:\\temp");
console.log(dx.getDownloadPath());

// <home> 문자열로 시작하면 사용자 폴더 하위 경로가 설정됩니다.
dx.setDownloadPath("<home>\\temp");
console.log(dx.getDownloadPath());

// <doc> 문자열로 시작하면 문서 폴더 하위 경로가 설정됩니다.
dx.setDownloadPath("<doc>\\temp");
console.log(dx.getDownloadPath());

// <desktop> 문자열로 시작하면 바탕화면 폴더 하위 경로가 설정됩니다.
dx.setDownloadPath("<desktop>\\temp");
console.log(dx.getDownloadPath());
