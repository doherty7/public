headerHeight Number

ex) headerGradientType: "HORIZONTAL"
headerColors String

헤더에 그레디언트 색상을 적용할 때 사용합니다.

헤더가 눌리지 않은 상태의 시작과 마지막 색, 헤더가 눌렸을 때 시작과 마지막 색상을 지정합니다.

ex) headerColors: "#FF0000 #00FF00 #FFFFFF #EFEFEF"

만일 그레디언트가 아닌 단일 색상을 지정하려면 시작과 마지막 색을 동일한 색상으로 지정합니다.
backgroundColor String

배경의 색상(16진수 표기법)을 설정합니다.

ex) backgroundColor: "#FF00FF"
backgroundImage String

배경의 가운데에 이미지를 표시합니다.

표시할 이미지의 웹경로, 대상 이미지의 가로 위치, 세로 위치를 지정합니다.

ex) backgroundImage: "http://이미지경로 50 50"

멀티 모듈과 달리 두 번째, 세 번째는 크기가 아닌 위치(%)입니다.
반환 값
설명

GUI 스타일을 변경합니다.

setUIStyle 함수에 스타일 속성이 지정된 json 객체를 설정하면 스타일이 변경됩니다.

멀티 모듈과 IE 모듈은 지원하는 속성이 다릅니다.

// 멀티 모듈 스타일을 변경합니다.
dx.setUIStyle({
 headerVisible: false, statusBarVisible: false, borderVisible: true
});

// IE 모듈 스타일을 변경합니다.
dx.setUIStyle({
 colorPattern: 1, imagePattern: 1, borderVisible: true, gridLineVisible: false
});
