openFolderDialog

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원 (2.0.1.0 버전부터 멀티 모듈에서는 제외됩니다.)
형식
isSelected = object.openFolderDialog();
파라미터
반환 값

폴더가 선택되었다면 true, 아니면 false를 반환합니다.

설명

업로드할 파일 및 폴더 구조를 추가하기 위한 폴더 선택 다이얼로그 창을 띄웁니다.

DEXTUploadX5는 컴포넌트 자체에 폴더를 추가하는 버튼이나 GUI 요소가 없습니다.

대신 스크립트를 사용하여 폴더를 추가할 수 있도록 함수를 제공하고 있습니다.

폴더 선택 다이얼로그 창에서 선택한 대상이 있다면 창이 닫힌 후, true 값을 반환합니다.

<button type="button" onclick="addFolder();">버튼</button>
<script>
function addFolder() {
  var dx = dx5.get("컴포넌트아이디");
  dx.openFolderDialog();
}
</script>

openFolderDialog 함수의 반환 값을 처리하는 코드가 없다면, 컴포넌트 생성 시점에 btnFolder 속성 값을 사용하여 자동으로 바인딩하여 사용할 수 있습니다. 

<button id="btn-add-folder" type="button">버튼</button>
<script>
  dx5.create({
    ...,
    // 폴더 추가 기능이 자동으로 연결됩니다.
    btnFolder: "btn-add-folder"
  });
</script>
