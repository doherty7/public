getCheckedItems

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
items = object.getCheckedItems([onlyLocal]);
파라미터
이름 형식 설명
onlyLocal Boolean [옵션] true이면 로컬 자원만 대상으로 삼습니다.
반환 값
체크 상태인 항목을 요소로 갖는 배열
설명

체크 상태인 항목을 요소로 갖는 배열을 반환합니다.

파라미터 값이 true이면 가상 파일을 제외한 로컬 자원만 카운트된 값이 반환됩니다.

// 체크된 항목들만 배열로 묶어 반환합니다.
var checkeds = dx.getCheckedItems();
