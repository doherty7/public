레퍼런스 > Component Object > Methods > setDecryptionSavePath

setDecryptionSavePath

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
object.setDecryptionSavePath(path);
파라미터
이름 형식 설명
path String

복호화된 파일 생성 경로

값 설명
드라이브 문자열로 시작 절대 경로
<X5TEMP>로 시작 DEXTUploadX5에서 정한 임시 경로
<WINTEMP>로 시작 Windows 임시 폴더
반환 값
설명

복호화된 파일 생성 경로를 설정합니다.

// 복호화 다이얼로그 타이틀을 설정합니다.
dx.setDecryptionDialogTitle("암호화 중...");

// 암호화된 파일을 가지고 복호화할 위치를 설정합니다.
// <X5TEMP>: DEXTUploadX5에서 정한 임시 위치
// <WINTEMP>: Windows 임시 폴더
// 드라이브 문자로 시작하는 경로: 대상 경로
dx.setDecryptionSavePath("c:\\temp");

// 복호화된 파일 생성되었는지 확인하기 위해, 탐색기 창을 엽니다.
dx.setDecryptionEnsuringFile(true);

// 복호화 작업을 시작합니다.
// 'AUTO', 'SELECTED', 'CHECKED' 옵션이 있습니다.
dx.decrypt("AUTO");
