exploreDownloadPath

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
object.exploreDownloadPath();
파라미터
반환 값
설명

설정된 다운로드 경로를 탐색기로 엽니다.

다운로드가 완료되면, 다운로드 결과를 보기 위해서 사용할 수 있습니다.

function onDX5DownloadCompleted(id) {
  dx5.get(id).exploreDownloadPath();
}
