getProgressType

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
type = object.getProgressType();
파라미터
반환 값

현재 설정된 진행창 종류

'0', '1', '2', '3'중 하나, 기본값은 '0'

설명

DEXTUploadX5 컴포넌트는 네 가지 형태의 진행창을 제공합니다.

dx5.create 함수로 컴포넌트를 생성할 때, progressType 속성 값을 '0', '1', '2', '3'로 설정할 수 있습니다.

멀티 모듈을 사용하여 ORAF 방식으로 업로드를 하는 경우, 파일 별 업로드 상태를 파악할 수 없는 관계로 '0'번 또는 '3'번 진행창만 사용이 가능합니다.

'3'번 진행창 형식은 파일 다운로드 작업을 수행할 때, 파일의 크기를 미리 알 수 없는 경우에 사용하도록 2.3.0.0 버전부터 추가되었습니다.

// '1'번 진행창을 사용합니다. 기본값은 '0'입니다.
dx5.create({ ..., progressType: "1" });
