getAuthkey

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
authkey = object.getAuthkey();
파라미터
반환 값
인증키 문자열
설명

인증키 문자열을 반환합니다.

인증키는 dextuploadx5-configuration.js 파일의 authkey 속성에 설정되어있습니다.

function onDX5Created(id) {
  var dx = dx5.get(id);
  // 인증키를 출력합니다.
  console.log(dx.getAuthkey());
}
