deleteMetaDataByIndex

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.deleteMetaDataByIndex(index, name);
파라미터
이름 형식 설명
index Number 항목의 순서
name String 메타 데이터의 이름
반환 값
설명

주어진 순서에 해당하는 항목의 메타 데이터를 삭제합니다.

항목은 "이름[구분자]값" 형식의 메타 데이터를 여러개 가질 수 있습니다.

메타 데이터는 업로드가 될 때, 서버로 전송이 되며, 순수 파일외에 다른 정보를 함께 전달하는 용도로 사용됩니다.

가상 파일은 업로드가 되지 않으므로 서버로 메타 데이터가 전달되지 않습니다.

메타 데이터를 삭제하려면 삭제할 메타 데이터의 이름이 필요합니다.

// 6번 째 항목에 메타 데이터를 설정합니다.
dx.setMetaDataByindex(5, "userid", "honggildong");
// "honggildong"이 출력됩니다.
console.log(dx.getMetaDataByIndex(5, "userid"));
// 메타데이터를 삭제합니다.
dx.deleteMetaDataByIndex(5, "userid");
