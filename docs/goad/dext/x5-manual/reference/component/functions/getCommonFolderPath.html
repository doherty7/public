getCommonFolderPath

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
folderPath = object.getCommonFolderPath(title);
파라미터
이름 형식 설명
title String 폴더 선택 다이얼로그 타이틀 명
반환 값
선택된 폴더 경로
설명

로컬 경로를 선택하기 위해 폴더 선택 다이얼로그 창을 띄웁니다.

getCommonFolderPath 함수는 DEXTUploadX5에 상태에 영향을 미치지 않는 유틸리티 기능입니다.

서비스 구현에 로컬 경로를 얻어야할 필요가 있을 때, 사용합니다.

var path = dx.getCommonFolderPath(title);
if (path) {
  ...
}
