setExtensionFilter

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.setExtensionFilter(filterString [, reverse]);
파라미터
이름 형식 설명
filterString String

확장자 필터링 처리를 위한 문자열

  • 순필터인 경우 예

    멀티 모듈: *.jpg;*.jpeg;*.gif;*.bmp;*.txt

    IE 모듈: 이미지파일(jpg,jpeg,gif,bmp)|*.jpg;*.jpeg;*.gif;*.bmp;|문서파일(txt)|*.txt;*||

  • 역필터인 경우 예

    멀티 모듈: *.jpg;*.jpeg;*.gif;*.bmp;*.txt

    IE 모듈: *.jpg;*.jpeg;*.gif;*.bmp;*.txt
reverse Boolean [옵션] true이면 역필터, false이면 순필터 문자열을 설정합니다.
반환 값
설명

확장자 필터링 처리를 위한 문자열 정보를 설정합니다.

DEXTUploadX5는 파일의 확장자 명을 기반으로 필터 처리가 가능합니다.

확장자 기반 필터 처리는 크게 순필터와 역필터로 나뉩니다.

  • 순필터: 필터에 명시된 확장자를 갖는 대상만 추가할 수 있습니다.
  • 역필터: 필터에 명시된 확장자를 갖는 대상은 추가할 수 없습니다.
/* 멀티 모듈 */
function onDX5Created(id) {
  var dx = dx5.get(id);
  // 순필터 적용 예제
  dx.setExtensionFilter("*.jpg;*.jpeg;*.gif;*.bmp;*.txt");
  // 역필터 적용 예제
  dx.setExtensionFilter("*.exe;*.dll;*.js;*.sh;*.com", true);
  // 설정된 확장자 필터 정보를 출력합니다.
  console.log("white => " + dx.getExtensionFilter());
  console.log("black => " + dx.getExtensionFilter(true));
}

/* IE 모듈 */
function onDX5Created(id) {
  var dx = dx5.get(id);
  // 순필터 적용 예제
  dx.setExtensionFilter("이미지파일(jpg,jpeg,gif,bmp)|*.jpg;*.jpeg;*.gif;*.bmp;|문서파일(txt)|*.txt;*||");
  // 역필터 적용 예제
  dx.setExtensionFilter("*.exe;*.dll;*.js;*.sh;*.com", true);
  // 설정된 확장자 필터 정보를 출력합니다.
  console.log("white => " + dx.getExtensionFilter());
  console.log("black => " + dx.getExtensionFilter(true));
}

IE 모듈용 순필터 문자열은 파이프 라인(|) 문자를 가지고 구분하며, 마지막에는 두 개의 파이프 라인 문자가 들어갑니다.

설명1|세미콜론으로 구분된 확장자 목록1|설명2|세미콜론으로 구분된 확장자 목록2||

setExtensionFilter 함수를 다시 호출하면, 기존 필터 정보는 제거되고 새로 덮어씁니다.
