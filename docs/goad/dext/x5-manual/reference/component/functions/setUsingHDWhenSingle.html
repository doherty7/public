setUsingHDWhenSingle

최소지원버전
2.0.0.0
지원환경
멀티 모듈 지원
형식
object.setUsingHDWhenSingle(enable);
파라미터
이름 형식 설명
enable Boolean true이면 HD 응용프로그램을 사용합니다. 기본값은 false입니다.
반환 값
설명

멀티 모듈에서 단일 파일 다운로드를 할 때 브라우저 다운로드가 아닌 HD 응용프로그램을 사용할 수 있도록 설정하거나 해지합니다.

화면상(GUI)에서 다운로드 아이콘을 클릭했을 때 실행되는 단일 파일 다운로드만 해당됩니다. downloadById 메소드를 사용하는 경우에는 설정과 상관 없이 브라우저 기반 다운로드가 수행됩니다.

// HD 응용프로그램을 사용하도록 설정합니다.
dx.setUsingHDWhenSingle(true);
