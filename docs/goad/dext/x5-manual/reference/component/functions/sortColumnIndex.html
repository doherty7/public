sortColumnIndex

최소지원버전
1.0.0.0
지원환경
멀티 모듈(1.1.0.0 이상), IE 모듈 지원
형식
object.sortColumnIndex(index, ascending);
파라미터
이름 형식 설명
index Number 정렬을 할 때, 기준이 되는 칼럼 순서
ascending Boolean true이면 오름차순, false이면 내림차순
반환 값
설명

주어진 칼럼을 기준으로 정렬을 합니다.

가상 파일을 최우선으로 한다면, 오름차순, 내림차순 상관없이 항상 가상 파일이 제일 먼저 위치합니다.

// 0번째 칼럼을 기준으로 오름 차순 정렬을 바로 수행합니다.
dx.sortColumnIndex(0, false);

IE 모듈은 0번째 칼럼이 파일 이름 또는 경로를 표현합니다. 그러나 멀티 모듈은 1번째 칼럼에 파일 이름이 명시됩니다. 그러므로 파일명으로 정렬할 때는 1, 파일 크기로 정렬할 때는 2를 index 값으로 설정해야 합니다. 만일 다른 값의 경우에는 정렬 작업을 수행하지 않습니다.

멀티 모듈의 tile 스타일의 경우에는 칼럼 순서에 상관없이 무조건 파일명을 기준으로 정렬을 수행합니다.

// 멀티 모듈이라면 파일의 크기를 기준으로 오름 차순 정렬을 수행합니다.
// 만일 타일 형식이라면 파일명을 기준으로 오름 차순 정렬을 수행합니다.
dx.sortColumnIndex(2, false);
