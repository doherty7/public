setDownloadPathOfHD

최소지원버전
2.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.setDownloadPath(path[, success]);
파라미터
이름 형식 설명
path String

다운로드 경로

값 설명
드라이브 문자열로 시작 절대 경로
<home>로 시작 사용자 폴더 하위 경로
<doc>로 시작 문서 폴더 하위 경로
<desktop>로 시작 바탕화면 폴더 하위 경로
success Function [옵션] 변경된 다운로드 경로 값을 얻을 수 있는 콜백 함수입니다.
반환 값
설명

다운로드 경로를 설정합니다.

// 드라이브 문자열로 시작하면 절대 경로가 설정됩니다.
dx.setDownloadPathOfHD("c:\\temp");

// <home> 문자열로 시작하면 사용자 폴더 하위 경로가 설정됩니다.
dx.setDownloadPathOfHD("<home>\\temp");

// <doc> 문자열로 시작하면 문서 폴더 하위 경로가 설정됩니다.
dx.setDownloadPathOfHD("<doc>\\temp");

// <desktop> 문자열로 시작하면 바탕화면 폴더 하위 경로가 설정됩니다.
dx.setDownloadPathOfHD("<desktop>\\temp");

만일 다운로드 경로를 변경한 후, 그 변경 값을 바로 알고 싶다면 콜백 함수(success)를 사용해야 합니다.

dx.setDownloadPathOfHD("<doc>\\temp", function (path) {
  console.log(path);
});

HD 응용프로그램을 사용하는 경우에 사용될 메소드들은 대부분 내부적으로 비동기 작업을 수행합니다. 비동기를 작업을 수행하기 때문에 작업 처리 결과를 받기 전까지 스레드가 대기하지 않으며, 요청에 대한 결과 값을 받기 위해서 콜백 함수를 이용해야 합니다.
