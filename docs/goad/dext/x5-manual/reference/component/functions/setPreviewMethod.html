setPreviewMethod

최소지원버전
1.2.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.setPreviewMethod(method);
파라미터
미리보기 방법. 1이면 내장 뷰어, 2이면 이벤트 호출 방식
반환 값
설명

미리보기 방법을 설정합니다.

1이면 내장 뷰어를 사용하여 이미지를 화면에 출력합니다.

2이며 이벤트를 호출하는 방식이 제공되며, 이 방법은 개발자가 자신 만의 미리보기 화면을 구성하는 용도로 사용할 수 있습니다.

이벤트 방식의 경우 브라우저에서 표현이 되어야 하기 때문에 IE 모듈이라고 하더라도 JPG, GIF, PNG, BMP 포맷으로 제한됩니다.

// true로 설정하면 뷰어를 사용할 수 있습니다.
dx.setPreviewEnable(true);

// 배경색을 설정합니다.
dx.setPreviewBackgroundColor("#ff0000");

// 이벤트 방식으로 설정합니다.
dx.setPreviewMethod(2);

// 첫 번째 파일이 로컬 이미지 파일이라면 내장 뷰어를 실행합니다.
dx.preview(0);

// 미리보기 이벤트가 발생하면 호출될 콜백함수입니다.
function onDX5Preview(id, itemIndex, itemId, itemSource) {
  ...
  if (itemSource) {
    target.src = itemSource;
  }
}
