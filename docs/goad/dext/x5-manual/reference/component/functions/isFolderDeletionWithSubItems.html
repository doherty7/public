isFolderDeletionWithSubItems

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
enable = object.isFolderDeletionWithSubItems();
파라미터
반환 값
true이면 하위 항목을 함께 삭제합니다. 기본값은 false입니다.
설명

폴더를 삭제할 때, 하위에 있는 항목들을 함께 삭제하는지를 반환합니다.

기본값은 false이며, 폴더를 삭제하더라도 하위 항목은 기본적으로 삭제되지 않습니다.

// 폴더를 삭제하면 하위 항목들도 삭제되도록 설정합니다.
dx.setFolderDeletionWithSubItems(true);
var enable = dx.isFolderDeletionWithSubItems();
console.log(enable);

사용자 액션에 의해 항목이 삭제될 때는 삭제 이벤트가 발생합니다.

폴더를 삭제할 때, 하위 항목들이 자동으로 삭제하도록 설정이 되었다면, 대상 폴더를 제외하고 하위 항목이 삭제될 때는 이벤트가 발생하지 않습니다.
