changeStatus

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
object.changeStatus(id, status);
파라미터
이름 형식 설명
id String 항목의 고유 아이디
status String "WAIT", "DONE"
반환 값
설명

주어진 고유 아이디에 해당하는 항목의 상태를 변경합니다.

로컬 자원은 업로드가 완료되면 status 값이 "DONE"으로 변경됩니다. status가 "DONE"인 대상은 다시 업로드가 되지 않습니다.

파일 업로드 작업이 없이 바로 "DONE"으로 변경하거나, 업로드가 완료된 대상을 "WAIT" 상태로 변경할 수 있습니다.

// 주어진 고유 아이디에 해당하는 항목을 "WAIT" 상태로 변경합니다.
dx.changeStatus(id, "WAIT");
