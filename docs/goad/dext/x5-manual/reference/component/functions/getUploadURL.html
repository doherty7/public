getUploadURL

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
url = object.getUploadURL();
파라미터
반환 값
서버 측 업로드 처리 주소
설명

서버 측 업로드 처리 주소를 반환합니다.

파일 업로드를 수행하려면 반드시 업로드 경로를 설정해주어야 합니다.

DEXTUploadX5는 RFC1867 규약에 따라 multipart/form-data 로 인코딩된 데이터를 서버로 전송하는 역할까지만 담당합니다.

서버 측 업로드 주소는 multipart/form-data 로 인코딩된 데이터를 받아 저장하는 서비스가 구현되어야 합니다.

function onDX5Created(id) {
  var dx = dx5.get(id);
  // 서버 측 업로드 처리 주소를 출력합니다.
  console.log(dx.getUploadURL());
}
