setLimitNetSpeed

최소지원버전
1.0.0.0
지원환경
IE 모듈 지원
형식
object.setLimitNetSpeed(limit);
파라미터
이름 형식 설명
limit Number 제한된 전송 속도 (bytes/sec)
반환 값
설명

전송 속도를 설정합니다.

파일 업로드 혹은 다운로드 시 초당 전송 데이터량을 조절할 수 있습니다.

setLimitNetSpeed 함수를 사용하면, 모든 업/다운로드 속도에 영향을 미칩니다.

주의) 제한이 없는 경우 네트워크에서 환경에 따라 속도가 달라집니다.

주의) 아무리 높은 속도로 설정한다고 하더라도 네트워크가 제공하는 최대치를 넘을 수 없습니다.

주의) 전송 속도는 설정한 값에 가깝도록 데이터 전송량을 조절하는 것으로써 설정값과 동일한 속도를 보장하는 것은 아닙니다.

// 전송 속도를 10KB 정도로 설정합니다. 
dx.setLimitNetSpeed(10240);
console.log(dx.getLimitNetSpeed());
