openFileDialog

최소지원버전
1.0.0.0
지원환경
멀티 모듈, IE 모듈 지원
형식
isSelected = object.openFileDialog();
파라미터
반환 값

파일이 선택되었다면 true, 아니면 false를 반환합니다.

멀티 모듈은 선택 여부를 확인할 수 없어 무조건 true가 반환됩니다.

설명

업로드할 파일을 추가하기 위한 파일 선택 다이얼로그 창을 띄웁니다.

DEXTUploadX5는 컴포넌트 자체에 파일을 추가하는 버튼이나 GUI 요소가 없습니다.

대신 스크립트를 사용하여 파일을 추가할 수 있도록 함수를 제공하고 있습니다.

파일 선택 다이얼로그 창에서 선택한 대상이 있다면 창이 닫힌 후, true 값을 반환합니다.

IE 모듈과 달리 멀티 모듈은 선택 여부를 확인할 수 없어 반환 값은 항상 true입니다.

멀티 모듈은 Internet Explorer 또는 MS Edge 브라우저에서 사용할 때, 파일 선택 다이얼로그 창을 사용하여 연속적으로 동일한 파일을 선택하는 경우, 파일이 추가되지 않는 현상이 있습니다. 

<button type="button" onclick="addFiles();">버튼</button>
<script>
function addFiles() {
  var dx = dx5.get("컴포넌트아이디");
  dx.openFileDialog();
}
</script>

openFileDialog 함수의 반환 값을 처리하는 코드가 없다면, 컴포넌트 생성 시점에 btnFile 속성 값을 사용하여 자동으로 바인딩하여 사용할 수 있습니다. 

<button id="btn-add-files" type="button">버튼</button>
<script>
  dx5.create({
    ...,
    // 파일 추가 기능이 자동으로 연결됩니다.
    btnFile: "btn-add-files"
  });
</script>
