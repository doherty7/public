DEXTUploadX5 소개

DEXTUploadX5는 웹 브라우저로부터 웹 서버로의 파일업로드 기능을 지원하는 HTTP 프로토콜 기반의 클라이언트 컴포넌트입니다.

DEXTUploadX5는 클라이언트 컴포넌트이기 때문에, 브라우저 위에서 동작하는 것을 원칙으로 하며, 서버로 파일을 보내거나, 내려받는 역할을 수행합니다.

DEXTUploadX5는 순수 웹 기술(HTML5, SVG) 만을 사용한 멀티 모듈과 고수준의 파일 업/다운로드 기능을 필요로 하는 서비스를 위하여 ActiveX 기술을 사용하는 Internet Explorer 전용 버전이 따로 존재합니다.

공통 기능
  • 한 번에 파일을 업로드하는 방식 지원 (ORAF)
  • 파일 단위로 업로드하는 방식 지원 (OROF)
  • 자사 서버컴포넌트와 연계하는 대용량 업로드 지원 (EXNJ) - 개별 2GB이상
  • 페이지가 다시 로드(load) 되어도 가능한 이어올리기 (대용량 업로드 사용시)
  • 개별/전체 용량 제한 설정
  • 전체 개수 제한 설정
  • 확장자 기반 필터링(순필터, 역필터)
  • 하나의 컴포넌트에서 업로드, 다운로드를 같이 처리
  • 멀티 컴포넌트 업로드 지원
  • 파일 업로드 결과(응답 데이터) 반환
  • 파일 드래그 앤 드롭
  • 파일별 메타 데이터 전송
  • 국제화를 위한 다국어 확장 지원
  • 기본 업로드 진행창 지원 (세 가지 타입)
  • 웹서버 인증, 쿠키, 세션 공유
  • SPA(Single-page Application) 서비스를 위한 함수 지원
  • 정렬 및 이동 (1.1.0.0부터 공통 지원)
  • 이미지 파일 미리보기 지원(1.2.0.0부터 공통 지원)
  • HD 응용프로그램을 이용한 파일 다운로드 (2.0.0.0부터 지원)
멀티 모듈
  • 플러그인 설치 불필요
  • 단일 파일 다운로드
  • 다중 파일 다운로드 (1.1.0.0부터 지원 - Safari 브라우저는 버전이 10.1 이상이고, OS가 macOS 경우만 지원)
  • 압축 다운로드 (1.3.0.0부터 지원)
  • 폴더 드래그앤 드롭 (1.1.0.0부터 크롬브라우저만 지원, 단 폴더 구조는 업로드되지 않습니다.)
IE 모듈
  • 다양한 다운로드 옵션 제공
  • 다중 파일 다운로드
  • 이어 받기 (서버 컴포넌트 연계)
  • 연결된 프로그램 실행 (파일 열기)
  • 폴더 구조 업로드
  • 폴더 드래그앤 드롭
  • 폴더 구조 다운로드
  • 암호화 전송 및 복호화 다운로드
  • 압축 전송
  • 자동 섬네일 생성
  • 클립보드 전송
  • 파일 업로드/다운로드 속도 제한
