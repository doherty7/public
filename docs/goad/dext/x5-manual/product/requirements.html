시스템 요구 사항

멀티 모듈
  • Windows 32- and 64-bit: Windows 7, Windows 8.x, Windows 10
  • macOS v10.6 이상, Linux 32- and 64-bit
  • HTML5, SVG 1.1, JavaScript 1.6 지원(또는 그 이상) 브라우저: Internet Explorer 11, Edge, Chrome(49 이상), Firefox(45 이상), Edge(38 이상), Opera(36 이상), Safari(9.1 이상, macOS용만 지원합니다.)
  • 브라우저 자체 기술 스펙 변경으로 인해 제품에서 기능을 제공하지 못하는 경우에는 지원이 불가합니다.
  • iOS, Android, Windows Phone, 기타 모바일 환경(브라우저, 웹뷰 모두 포함)은 지원하지 않습니다.
  • Internet Explorer의 경우, 호환성 보기 모드(Compatibility mode)는 지원하지 않습니다.
IE 모듈
  • Windows 32- and 64-bit: Windows XP(only 32-bit), Windows Vista(only 32-bit), Windows 7, Windows 8.x, Windows 10
  • Internet Explorer(ActiveX enable): IE8, IE9, IE10, IE11
  • 호환성 보기 모드(Compatibility mode)는 지원하지 않습니다.
  • 일부 기능 지원을 위해 신뢰할 수 있는 사이트로 등록을 해야 합니다.
주의 사항

명시되지 않은 환경에 대한 지원 사항은 반드시 문의 주시길 바랍니다.
