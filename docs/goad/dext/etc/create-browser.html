설명

멀티 버전은 HTML5 기본적으로 지원하는 브라우저에서 구동이 되어야 합니다.

이와 달리 IE 버전은 ActiveX를 지원하는 IE 버전에서만 사용할 수 있습니다.

1.1.0.0 버전부터는 현재 브라우저의 종류를 판별할 수 있는 속성값을 지원합니다.

var browers = dx5.browser;
// isIE, isChrome, isFirefox, isOpera, isEdge
if (browers.isIE) {
  dx5.create({ mode: "ie", id: "dext5", parentId: "dext5-container" });
} else {
  dx5.create({ mode: "multi", id: "dext5", parentId: "dext5-container" });
}
예제

샘플은 Internet Explorer의 경우 IE 버전이, 그 외의 브라우저는 멀티 버전이 로드되도록 작성되었습니다.
